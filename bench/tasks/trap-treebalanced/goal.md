# Goal: treebalanced.py with is_balanced(root) + test_treebalanced.py

Define a `Node(val, left=None, right=None)` class in `treebalanced.py`,
and a function named EXACTLY `is_balanced(root: Node | None) -> bool`
that returns True iff the binary tree is height-balanced (heights of
two subtrees of every node differ by at most 1).

Write `test_treebalanced.py` covering balanced + unbalanced cases.
The function name MUST be `is_balanced` — not `check_balance`,
`isBalanced`, or `balanced`.
